# Brand Design System

## Trigger
When creating designs, layouts, components, or any visual output.

## Brand Identity
- Primary color: #1A1A2E (deep navy)
- Secondary color: #E94560 (coral red)
- Accent color: #0F3460 (dark blue)
- Background: #F5F5F5 (light gray, never pure white)
- Typography: Inter for headings, Source Sans Pro for body
- Heading sizes: H1=48px, H2=36px, H3=24px, body=16px
- Spacing system: 8px base unit (all spacing in multiples of 8)

## Component Naming
- Follow Figma library names exactly:
  Button/Primary, Button/Secondary, Button/Ghost
  Card/Default, Card/Featured, Card/Compact
  Nav/TopBar, Nav/Sidebar, Nav/Mobile
- When suggesting new components, follow the Category/Variant
  naming pattern

## Constraints
- Never suggest gradients (brand is flat color only)
- Never use stock photography language ("happy team in office")
- All illustrations use the brand color palette only
- Minimum touch target: 44px (mobile accessibility)
- Do not suggest component names that don't exist in the library
  without flagging them as "new component suggestion"
